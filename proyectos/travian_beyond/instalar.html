GreaseMonkey

Con GreaseMonkey debes usar el navegador Firefox

Para instalar la extension GreaseMonkey en Firefox:

  1. Dirigete a este enlace.
  2. Selecciona "Install Now"
  3. Te aparecera un formulario con el boton "Instalar ahora"
  4. Reinicia Firefox

Y ahora para instalar el script:

  1. Vuelve a la pagina principal del script
  2. Sobre el enlace de la descarga pulsa el boton derecho del raton para sacar el menu contextual y selecciona "Install User Script...".
  3. A los pocos segundos aparecera un mensaje de confirmacion

Ya esta listo para usar. Para futuras actualizaciones del script, simplemente repite estos 3 ultimos pasos

Para instalar el script en Opera 9.0

  1. Descarga el fichero del script y ponlo en algun directorio de tu disco duro, por ejemplo, C:\UserJS\
  2. Dentro de Tools -> Preferences -> Advanced -> Content -> Javascript Options escoje en User Javascript Files la carpeta que has creado en el punto anterior

Ya esta listo para usar. Para futuras actualizaciones del script, simplemente sustituye el fichero en ese directorio

NOTA: En algunos casos para Opera, es posible que el fichero haya que renombrarlo a "travian.js" sin el "user"
