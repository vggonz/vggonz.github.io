Ogame Map

Utilidad de cartografia para el juego de navegador Ogame:

  • Acceso a base de datos
    • Herramientas genericas
    • Fuente del acceso a MySQL
    • Fuente de las funciones de paginacion de resultados
  • Indexacion en base de datos
    • Ejecucion manual por linea de comandos
    • Necesita la cookie de sesion que se encuentra en todos los enlaces del juego
    • Ejemplo: "php admin_cmd.php session universo galaxia1 galaxia2 sistema1 sistema2" indizaria los sistemas comprendidos entre sistema1 y sistema2 para todas las galaxias entre galaxia1 y galaxia2 en el servidor de juego del universo 18.
    • Solo funciona con los servidores hispanos de juego accesibles desde http://ogame.com.es
    • Guarda la fecha y hora de inseracion para conocer la frescura de los datos
    • Fuente
    • Descripcion SQL de la lista de universos conocidos
    • Descripcion SQL de la tabla de los planetas de un servidor
    • Descripcion SQL de la tabla de los jugadores de un servidor
    • Descripcion SQL de la tabla de alianzas de un servidor
    • Buscador
      • Busca entre los resultados de la indexacion
      • Parametrizado por etiqueta de la alianza, nombre del jugador, nombre del planeta, posicion, ranking y estado (inactivo, vacaciones, etc)
      • Resultados paginados
      • Fuente
    • Userscript
      • Script para los informes y mensajes
      • Muestra para cada informe de espionaje el robo teorico a partir de los recursos
      • Calcula las naves de carga necesarias para recogerlos y su porcentaje de carga
      • A partir de las naves en un planeta, calcula los escombros teoricos que producirian y los recicladores necesarios para recogerlos junto con su porcentaje de carga
      • Para todos los mensajes muestra un enlace para ocultar o desplegar el cuerpo de cualquier mensaje
      • Adaptado para la version 0.73e (aunque puede que funcione para otras versiones del juego)
      • Solo valido para los servidores hispanos de http://ogame.com.es
      • Solo funciona con Firefox y GreaseMonkey (no con Opera)
